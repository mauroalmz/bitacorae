TechScan PRO · Diagnóstico de hardware profesional y portable

🔧 Cliente .exe · Servidor central · Dashboard web · 100% personalizable

TechScan PRO es un sistema completo para técnicos de soporte informático. Permite ejecutar diagnósticos de hardware desde un simple pendrive, sin instalar nada en la PC del cliente. Los resultados se envían a un servidor central accesible vía web, donde supervisores y administrativos pueden visualizar el estado de los equipos, gestionar técnicos y llevar un historial permanente.

⚙️ ¿Cómo funciona?

  • Cliente portable (.exe): se ejecuta en Windows 10/11, no requiere instalación. Realiza pruebas de estrés de CPU/RAM, verificación de disco (SMART), test de red, recolección de datos del sistema (serial, modelo, BIOS).
  • Servidor central: corre en cualquier PC de la red (Ubuntu/Windows) con Python + Flask. Recibe heartbeats y reportes, y sirve un dashboard web actualizado en tiempo real.
  • Almacenamiento: archivos CSV y JSON (fáciles de respaldar, auditar o exportar a sistemas externos).

🎯 Personalización para cada empresa

TechScan PRO se adapta totalmente a tu identidad y necesidades:

  • Branding completo: reemplazamos logos, íconos, colores y nombre del ejecutable.
  • Pruebas adicionales: integración con herramientas externas (AIDA64, FurMark) o desarrollo de tests personalizados.
  • Adaptación del dashboard: diseño con la paleta de tu empresa, módulos a medida.
  • Integración con sistemas de tickets: API para conectar con tu backend actual.

📊 Funcionalidades clave (versión 1.0)

  • Diagnóstico de CPU, RAM, disco (SMART), red y latencia.
  • Recolección de datos del sistema (serial, modelo, BIOS, temperatura).
  • Dashboard web con visualización en tiempo real de equipos activos y su estado.
  • Historial completo por número de serie del equipo.
  • Gestión de técnicos (alta, baja, reseteo de contraseñas).
  • Reportes en CSV con trazabilidad (fecha, técnico, resultados, observaciones).

🔒 Seguridad y control

  • Contraseñas con hash PBKDF2.
  • Clave maestra de administrador para resetear cuentas.
  • Los técnicos solo ven los equipos que ellos mismos registran.
  • El servidor puede alojarse en una red interna sin exposición a internet.
